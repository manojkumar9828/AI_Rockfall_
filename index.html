About The Project

Protecting miners through artificial intelligence

Project Overview

Rockfall is one of the most critical hazards in open pit mining, responsible for equipment damage, production delays, and fatal accidents. Traditional monitoring relies on visual inspection by geotechnical engineers, which is subjective, slow, and limited by human availability.

This project develops an AI-powered system using a Convolutional Neural Network (CNN) trained on mine slope images to classify rock conditions into Safe, Warning, and Dangerous categories. When dangerous conditions are detected, the system generates immediate evacuation alerts, enabling proactive safety management.

Technology

  • TensorFlow CNN Model
  • Flask Web Backend
  • Bootstrap 5 UI
  • Chart.js Analytics
  • SQLite Database
  • OpenCV Processing

Project Objectives

Early Detection

Detect unstable rock formations before rockfall occurs using CNN image classification.

Instant Alerts

Generate automatic evacuation alerts when dangerous conditions are identified.

Data Analytics

Maintain prediction history and generate visual reports for safety analysis.

User Management

Role-based access for administrators and field operators with secure sessions.

Key Features

Image Upload

Upload JPG, JPEG, or PNG images of mine slopes through a simple drag-and-drop interface with instant preview.

AI Prediction

CNN model classifies images into Safe, Warning, or Dangerous with confidence scores in real time.

Alert System

Color-coded alerts: red for evacuation, orange for warning, green for safe conditions.

Alert History

Every prediction is stored in SQLite with search, delete, and CSV export capabilities.

Reports

Generate daily, weekly, and monthly reports with charts, statistics, and accuracy metrics.

Admin Panel

Manage users, view prediction history, monitor system logs, and oversee all reports.
